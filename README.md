# Abouts
This is a demo app of student and teacher panel. Though this app only works for one mobile and the UI is not
so standard but the functionaly I worked on is sufficient to handle teacher and student communication.
# Features
- Student Login
- Teacher controlling over classroom including add a student or remove a student
- Updates of notice
- Updates of students result sheet
- Class time and Link update
- Managing Blood from the students at emergency situation
# Tools Used
1) SQLite database
2) Manual Authentication using existing SQLite database
3) Android Studio and xml
